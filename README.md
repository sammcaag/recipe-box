# Welcome to RecipeBox!

## Features

- Browse to a list of Recipes
- Add And Delete Recipes with ease
- You can also add favorites whenever you want!

## Getting Started

### Installation

Clone my application:

```bash
git clone https://github.com/sammcaag/recipe-box
```

Run npm install first before strating
```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

### List of shadcn/ui components used:
- button
- card
- dialog
- input
- label

Recipe Box application will be available at `http://localhost:5173`.

### Data Source

My data is from recipe.json
